---
title: 'Digital Asset Standard (DAS) API'
sidebarTitle: 'Overview'
description: 'This page provides an overview of all available Digital Asset Standard (DAS) API endpoints for working with tokens, NFTs, and compressed NFTs on Solana.'
---

<CardGroup cols={2}>
  <Card title="getAsset" href="/das-api/getasset">
    Retrieves information about a specific asset by its ID.
  </Card>
  <Card title="getAssetBatch" href="/das-api/getassetbatch">
    Retrieves information about multiple assets in a single request.
  </Card>
  <Card title="getAssetProof" href="/das-api/getassetproof">
    Fetches a proof for a specific compressed asset.
  </Card>
  <Card title="getAssetProofBatch" href="/das-api/getassetproofbatch">
    Fetches proofs for multiple compressed assets in a single request.
  </Card>
  <Card title="getAssetsByAuthority" href="/das-api/getassetsbyauthority">
    Returns assets by the specified authority address.
  </Card>
  <Card title="getAssetsByCreator" href="/das-api/getassetsbycreator">
    Retrieves assets created by a specific creator address.
  </Card>
  <Card title="getAssetsByGroup" href="/das-api/getassetsbygroup">
    Returns assets that belong to a specific group.
  </Card>
  <Card title="getAssetsByOwner" href="/das-api/getassetsbyowner">
    Lists all assets owned by a specific address.
  </Card>
  <Card title="getNftEditions" href="/das-api/getnfteditions">
    Retrieves information about NFT editions.
  </Card>
  <Card title="getSignaturesForAsset" href="/das-api/getsignaturesforasset">
    Returns transaction signatures involving a specific asset.
  </Card>
  <Card title="getTokenAccounts" href="/das-api/gettokenaccounts">
    Retrieves token accounts based on specified parameters.
  </Card>
  <Card title="searchAssets" href="/das-api/searchassets">
    Searches for assets based on various criteria.
  </Card>
</CardGroup>
