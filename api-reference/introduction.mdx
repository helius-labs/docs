## Core Features

Helius offers access to Solana blockchain data through:

- **Standard RPC Methods**: Access all standard Solana RPC functionality with improved reliability and performance
- **Real-time Websockets**: Establish persistent connections for immediate updates without polling
- **DAS Compliance**: Fully compliant with Solana's Digital Asset Standard for NFT metadata

## Endpoints

Helius provides multiple endpoint types to suit different application needs. For a comprehensive list of all available endpoints and their connection details, see our [Endpoints](/api-reference/endpoints) documentation.

## Authentication

All Helius API endpoints require authentication using an API key. Include your API key as a query parameter (`api-key=YOUR_API_KEY`) with every request.

More information on authentication can be found in the [Authentication](/api-reference/authentication) section.
