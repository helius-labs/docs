## Available APIs

Helius offers multiple APIs for reading data, each suited for different use cases:

### Standard Solana RPC API

The core JSON-RPC API provides access to all raw data on the Solana blockchain. Key methods include:
- `getAccountInfo`
- `getProgramAccounts`
- `getTransaction`

### DAS API (Digital Asset Standard)

A specialized API for working with NFTs and tokens that provides:
- Rich metadata retrieval
- Collection and ownership queries
- Advanced filtering capabilities

### Enriched Transaction API

Makes Solana transactions human-readable by:
- Parsing transaction instructions
- Identifying token transfers
- Categorizing transaction types

### WebSockets and LaserStream

For real-time data and subscription-based patterns:
- Account updates
- Transaction confirmations
- Solana program logs
