## 购买节点

<Accordion title="专用节点位于哪里？">
**北美**：匹兹堡、纽瓦克、盐湖城、洛杉矶、温哥华

**欧洲**：都柏林、伦敦、阿姆斯特丹、法兰克福

**亚洲**：东京、新加坡
</Accordion>

<Accordion title="新专用节点何时可用？">
加入我们的 [Discord](https://discord.com/invite/6GXdee3gBj)、[Telegram](https://t.me/helius_help) 支持频道，并关注我们的 [Twitter](https://x.com/heliuslabs) 以获取更新。
</Accordion>

<Accordion title="购买专用节点后可以启用 gRPC 吗？">
可以——打开一个 [工单](https://dashboard.helius.dev/support)，我们会为您启用它。
</Accordion>

<Accordion title="专用节点是否具有 Solana 存档访问权限？">
不——专用节点提供大约 48 小时的历史记录。使用标准计划获取较旧的数据。
</Accordion>

## 管理节点

<Accordion title="我可以更换专用节点的 API 密钥吗？">
可以——提供您的主机 ID，[支持](https://www.helius.dev/docs/support) 团队将为您提供新密钥。
</Accordion>

<Accordion title="迁移到 LaserStream 后我应该保留我的专用节点吗？">
通常不需要。LaserStream 加上共享 RPC 可以满足大多数需求。
</Accordion>

## 故障排除节点

<Accordion title="为什么我的节点不工作？">
检查您的仪表板中的节点状态并验证您的端点 URL 是否正确。常见问题包括网络连接、错误的 API 密钥或临时节点维护。如果问题仍然存在，请 [联系支持](https://www.helius.dev/docs/support) 并提供您的节点详细信息。
</Accordion>

<Accordion title="为什么 gRPC 停止工作？">
gRPC 可能在节点维护期间或由于资源限制而被禁用。检查您的仪表板以获取节点状态，验证您的 gRPC 配置，并确保您的客户端使用正确的连接参数。如果问题仍然存在，请 [联系支持](https://www.helius.dev/docs/support)。
</Accordion>

<Accordion title="为什么我的专用节点不健康？">
常见原因包括操作系统自动更新、重启或内存不足事件。在节点恢复期间，您将自动切换到备用端点。
</Accordion>
