Solana 账户在区块链上存储所有原始数据。本指南演示如何使用 Helius API 高效检索账户数据。

## 使用 RPC 方法

读取账户数据的最直接方法是通过 Solana RPC 方法：

```typescript
// Replace YOUR_API_KEY with the API key from your Helius dashboard
const response = await fetch(`https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY`, {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    jsonrpc: "2.0",
    id: "1",
    method: "getAccountInfo",
    params: ["83astBRguLMdt2h5U1Tpdq5tjFoJ6noeGwaY3mDLVcri", { encoding: "jsonParsed" }],
  }),
});
const data = await response.json();
console.log("Account data:", data.result.value);
```

## 常用参数

- `encoding`: 从 `base58`, `base64` 或 `jsonParsed` 中选择（推荐）
- `commitment`: 数据一致性级别（`processed`, `confirmed` 或 `finalized`）
- `dataSlice`: 仅获取特定部分的账户数据

## 获取多个账户

对于批量检索，使用 `getMultipleAccounts`:

```typescript
const response = await fetch('https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    jsonrpc: '2.0',
    id: '1',
    method: 'getMultipleAccounts',
    params: [
      ['83astBRguLMdt2h5U1Tpdq5tjFoJ6noeGwaY3mDLVcri', 'SysvarC1ock11111111111111111111111111111111', 'TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA'],
      { encoding: 'jsonParsed' }
    ]
  })
});
const data = await response.json();
console.log("Multiple accounts data:", data.result.value);
```

## 最佳实践

- 通过 WebSockets 使用账户订阅以获取实时更新
- 对于大规模数据需求，考虑使用 Helius DAS API 或 LaserStream
- 在适当的时候缓存账户数据以减少 API 调用
