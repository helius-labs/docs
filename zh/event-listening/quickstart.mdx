本指南将引导您使用 Helius 设置您的第一个事件监听器。

## 先决条件

- 一个 Helius API 密钥（从 [Helius Dashboard](https://dashboard.helius.dev) 获取）
- 一个可以接收 POST 请求的 webhook 端点（用于 webhook 监听器）

## 设置选项

Helius 提供两种主要的事件监听方法：

1. **Webhooks** - 当特定区块链事件发生时接收通知
2. **WebSockets** - 用于响应式应用程序的实时事件流
