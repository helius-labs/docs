人们使用 [Solana 浏览器](https://www.helius.dev/blog/top-solana-block-explorers) 的一个常见原因是通过输入代币的地址来查找 Solana 代币的详细信息，这通常被称为“合约地址”（CA），在 Solana 上更准确地称为代币铸造账户。

使用 [Orb](https://orb.helius.dev)，交易者可以探索代币的价格、交易量、持有者数量、流通代币供应、市场、流动性、代币元数据、权限，并访问交易图表。

在下面的示例中，我们将查看 [\$ORE 的 Solana 代币页面](https://orb.helius.dev/address/oreoU2P8bN6jkk3jbaiVxYnG1dCXcYxwhwyK9jSybcp/history)：

地址: (`oreoU2P8bN6jkk3jbaiVxYnG1dCXcYxwhwyK9jSybcp`)

## Solana 代币概览

在每个代币页面的顶部，您可以看到最重要的指标，包括：

- 代币名称、图标和地址
- 网站和社交媒体链接
- 代币总供应量和市值
- 验证状态和交易图表链接
- 代币价格、持有者数量和交易量

## 代币交易历史

**历史** 标签显示代币最近活动的列表，包括交易类型、摘要、执行时间、调用的程序和交易签名。

<Frame caption="\$ORE 的实时代币交易动态，显示最近的交换、铸造和转账。">

  ![Ore 的实时代币交易动态，显示最近的交换、铸造和转账](/images/orb/explore-tokens/solana-token-transaction-history-detail-page-orb-explorer.jpg)

</Frame>

选择“过滤器”会打开一个面板，提供按以下选项过滤的功能：

- 状态（例如，成功/失败）
- 代币（例如，USDC）
- 程序（例如，DFlow Aggregator v4）

<Tip>
  您还可以隐藏垃圾信息，并按最早排序结果，以查看代币的最早交易。
</Tip>

## 代币市场

点击 **市场** 标签查看所有流动性对（例如，ORE-SOL，ORE-USDC）、交易场所（例如，Orca，Meteora DLMM）以及包括流动性、24小时交易量、总交易次数和独立交易者数量在内的指标。

<Frame caption="Solana 上 $ORE 的顶级市场和流动性池，包括 ORE-SOL 和 ORE-USDC。">

  ![Solana 上 Ore 的顶级市场和流动性池，包括 ORE-SOL 和 ORE-USDC](/images/orb/explore-tokens/solana-token-markets-data-on-the-orb-block-explorer.jpg)

</Frame>

## 代币持有者

交易者可能希望验证代币的供应是否没有高度集中在少数持有者手中。点击 Orb 中的 **持有者** 标签，可以快速分析顶级钱包、其供应百分比和价值（以美元计价）。

<Frame caption="ORE 的代币持有者列表、代币数量、百分比和持有价值。">

  ![Ore 的代币持有者列表、代币数量、百分比和持有价值](/images/orb/explore-tokens/solana-token-holder-details-page-on-the-orb-explorer.jpg)

</Frame>

## 代币元数据

要查看或复制任何 Solana 代币的元数据，请点击 Orb 中的 **元数据** 标签。

JSON 视图提供了代币的关键信息，包括链接、描述、版税、所有权详情和代币信息（例如，代码、供应量、小数位和铸造权限）。

<Frame caption="ORE 的 JSON 代币元数据，包括文件、描述和链接。">

  ![Ore 的 JSON 代币元数据，包括文件、描述和链接](/images/orb/explore-tokens/solana-token-metadata-json-on-the-orb-explorer.jpg)

</Frame>

## 代币权限

权限定义了 Solana 上代币的控制和权限（例如，铸造新代币、冻结账户和修改属性）。

点击 **权限** 标签，我们可以看到 ORE 的代币权限及其影响。

<Frame caption="代币权限页面显示 ORE 的权限、铸造权限、代币程序和总供应量。">

</Frame>
