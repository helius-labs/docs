像 [Orb](https://orb.helius.dev) 这样的区块浏览器是[寻找顶级 Solana 代币](https://www.helius.dev/orb/explore-markets)、研究项目以及决定是否买卖代币的绝佳工具。

为了使这个过程更加简单，我们在 Orb 浏览器中每个 Solana 代币页面上添加了一个“购买”按钮。您只需连接您的钱包并点击“购买”即可在 Solana 上的任何 SPL 代币之间进行交换。

本教程解释了如何使用 Orb 在 Solana 上交换代币，并提供了用于更快研究和更轻松购买的键盘快捷键。

## **在 Orb 上交换 Solana 代币**

以下是关于如何在 Orb 上查找和交易代币的分步教程。

### **查找代币**

首先，通过在 Orb 的搜索栏中按名称搜索或输入代币的铸造地址来查找您要交换的代币。

<Frame caption="Orb 代币搜索栏显示 Solana 上的 ZEC 代币">

  ![使用 Orb 浏览器在 Solana 上查找 Zec 代币](/images/find-zec-token-on-solana-using-the-orb-explorer.jpg)

</Frame>

了解有关[查找 Solana 代币](https://www.helius.dev/docs/orb/explore-tokens)和[查找 SPL 铸造地址](https://www.helius.dev/docs/orb/explore-authorities)的更多详细信息。

### 点击“购买”

在页面顶部，代币的图像和名称旁边，点击橙色的“购买”按钮。

您还将看到关键的代币指标，包括供应量、市值、流动性、完全稀释价值（FDV）、当前价格、持有者（在 Solana 上）和 24 小时交易量。

<Frame caption="Orb 浏览器上的 Solana 代币页面，突出显示了“购买”按钮">

  ![Zec Token Explorer And Swap Button On Orb Jp](/images/zec-token-explorer-and-swap-button-on-orb.jpg)

</Frame>

在这个例子中，我们将在 Solana 上将 SOL 兑换为 ZEC（Zcash）。

### 连接您的钱包

点击“购买”按钮或按下键盘快捷键“B”会打开一个对话框，提示您连接钱包。

<Frame caption="连接钱包的对话框，由 reown 提供支持">

  ![Connect Solana Wallet To The Orb Explorer To Swap Tokens Jp](/images/connect-solana-wallet-to-the-orb-explorer-to-swap-tokens.jpg)

</Frame>

### 使用交换小部件

连接后，您将看到主要的交换小部件。

<Frame caption="Orb 交换小部件，指示 0.015 SOL 交换为 ZEC">

  ![The Orb Explorer Swap Widget For Trading Tokens On Solana Jp](/images/the-orb-explorer-swap-widget-for-trading-tokens-on-solana.jpg)

</Frame>

此交换小部件 UI 允许您选择要买卖的资产和数量。

一旦您点击“购买 ZEC”，您的钱包将显示确认消息。

<Frame caption="Phantom 交换确认消息，用于在 Orb 上交易">

  ![Phantom Wallet Swap Confirmation For Trading On The Orb Explorer Jp](/images/phantom-wallet-swap-confirmation-for-trading-on-the-orb-explorer.jpg)

</Frame>

一旦您点击确认，您的交易将通过 [DFlow](https://dflow.net/) 的元聚合器或 Jupiter 路由。DFlow 的智能交易执行堆栈为交易者提供最佳价格和最小价差。

<Info>
  所有通过 Orb UI 进行的交换都会收取 0.1% 的交换费。
</Info>

### 查看交换详情

交换完成后，您将看到交换确认消息和一个链接以[查看交易详情](https://www.helius.dev/docs/orb/explore-transactions)。点击此链接查看您的交易摘要。

<Frame caption="Orb 探索器上的交换交易摘要页面">

  ![Sol Zec Swap Transaction Explorer Summary On The Orb Jp](/images/sol-zec-swap-transaction-explorer-summary-on-the-orb.jpg)

</Frame>

就是这样！

您已成功在 Orb 上将 Solana 兑换为 ZEC。

要[查看您的钱包余额](https://www.helius.dev/docs/orb/explore-wallets)，请前往 Orb 上的钱包页面并点击**代币**标签。

<Frame caption="在 Orb 上探索任何钱包的代币余额">

  ![Orb 上的 Solana 钱包浏览器标签](/images/solana-wallet-explorer-tab-on-orb.jpg)

</Frame>
