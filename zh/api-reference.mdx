### 代币与交易 API

<CardGroup cols={2}>
  <Card title="数字资产标准 (DAS)" icon="hexagon-image" href="/zh/api-reference/das">
    通过单个 API 调用访问标准化的代币和 NFT 元数据。自动处理常规和压缩的 NFT。
  </Card>
  <Card title="优先费用 API" icon="star" href="/zh/api-reference/priority-fee/getpriorityfeeestimate">
    根据当前网络状况获取推荐的交易费用。防止超时并确保交易的及时确认。
  </Card>
  <Card title="增强交易" icon="receipt" href="/zh/api-reference/enhanced-transactions">
    以人类可读的格式检索预解析的交易数据。通过结构化信息节省开发时间，准备好显示。
  </Card>
  <Card title="ZK 压缩" icon="arrows-to-circle" href="/zh/api-reference/zk-compression">
    将链上数据的账户存储成本显著降低至 98%。
  </Card>
</CardGroup>
