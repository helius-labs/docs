<Info>
**Helius AirShip** 是一个开源工具，使代币空投变得简单、经济且易于访问。无论您是向 Saga Chapter 2 持有者、NFT/cNFT 持有者、SPL 代币持有者分发，还是使用 CSV 文件，AirShip 都能简化流程，并通过 ZK 压缩大幅降低成本。
</Info>

## 什么是 AirShip？

AirShip 是一个全面的空投解决方案，利用 ZK 压缩使大规模代币分发在 Solana 上变得经济实惠。AirShip 不再为每个接收者创建单独的代币账户（每个账户约需 0.002 SOL），而是使用 ZK 压缩将接收者数据捆绑到一个单一的 Merkle 根中，从而将成本降低超过 95%。

<Note>
有关空投成本和 ZK 压缩工作原理的详细说明，请阅读我们的综合指南：[The Cheapest Way to Airdrop Solana Tokens](https://www.helius.dev/blog/solana-airdrop)
</Note>

## 使用网页版

<Info>
网页版非常适合较小的空投和喜欢图形界面的用户。
</Info>

<Steps>
  <Step title="访问 AirShip">
    访问 [AirShip 网站](https://airship.helius.dev) 并点击 "Create New Airdrop"。
  </Step>
  
  <Step title="配置您的钱包">
    1. 输入您的私钥和 RPC URL
    2. 使用一个 RPC，比如 `https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY`
  </Step>

    <Warning>
    使用专为此空投任务创建的临时钱包。私钥仅用于签署交易，不会存储或共享。
    </Warning>
  </Step>
  
  <Step title="选择接收者">
    从多个选项中选择：
    
    <CardGroup cols={2}>
      <Card title="NFT 持有者" icon="image">
        选择特定的 NFT 集合
      </Card>
      <Card title="代币持有者" icon="coins">
        目标是特定 SPL 代币的持有者
      </Card>
      <Card title="Saga 持有者" icon="mobile">
        分发给 Saga 第 2 章的持有者
      </Card>
      <Card title="CSV 上传" icon="file-csv">
        上传您自己的接收者列表
      </Card>
    </CardGroup>
  </Step>
  
  <Step title="设置代币数量">
    指定以下之一：
    - 每个接收者的固定代币数量
    - 分发的总代币供应的百分比
  </Step>
  
  <Step title="审核并确认">
    仔细审核空投摘要：
    - 总接收者数
    - 每个接收者的代币数量
    - 预计交易费用
    - 总成本

    一旦确认，AirShip 将自动处理批量发送和 ZK 证明。
  </Step>
</Steps>

## 使用 CLI 版本

<Info>
CLI 版本更适合较大的空投和喜欢命令行工具的用户。
</Info>

### 安装

<Tabs>
  <Tab title="npm">

    ```bash
    npm install -g helius-airship
    ```

  </Tab>
  <Tab title="从源码构建">

    ```bash
    git clone https://github.com/helius-labs/airship.git
    cd airship
    pnpm install && pnpm build
    cd packages/cli
    pnpm link --global
    ```

  </Tab>
</Tabs>

### 使用方法

<CodeGroup>

```bash Basic Command
helius-airship \
  --keypair /path/to/your/airdrop_wallet.json \
  --url "https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY"
```

```bash With Help
helius-airship --help
```

</CodeGroup>

<Note>
将 `/path/to/your/airdrop_wallet.json` 替换为您的 Solana 文件系统钱包路径，并将 `YOUR_API_KEY` 替换为您的 Helius API 密钥。
</Note>

### 交互式设置

<Steps>
  <Step title="选择接收者类型">
    从 NFT 持有者、代币持有者、CSV 等中选择
  </Step>
  <Step title="指定代币数量">
    设置每个接收者的分发数量
  </Step>
  <Step title="确认详情">
    查看空投配置
  </Step>
  <Step title="执行交易">
    AirShip 将处理批量处理
  </Step>
</Steps>

## 要求

<CardGroup cols={3}>
  <Card title="Node.js" icon="node-js">
    Node.js >= v20.9.0
    
    pnpm >= v9.13.2（用于从源码构建）
  </Card>
  <Card title="Solana 钱包" icon="wallet">
    一个有效的文件系统钱包，需满足：
    - 持有要分发的代币
    - 有用于交易费用的 SOL
    - 拥有正在分发的代币
  </Card>
  <Card title="RPC 端点" icon="server">
    必须支持：
    - ZK 压缩
    - 数字资产标准 (DAS) API
    
    [在 Helius Dashboard 免费获取](https://www.dashbord.helius.dev)
  </Card>
</CardGroup>

<Note>
如果您没有文件系统钱包，请访问 [Solana 文档](https://docs.solana.com/cli/install-solana-cli-tools) 了解创建方法。
