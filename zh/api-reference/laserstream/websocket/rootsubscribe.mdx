title: rootSubscribe

## 端点

主网和开发网提供 Websockets，使用以下 URL：

- **主网** `wss://laserstream-mainnet.helius-rpc.com/?api-key=<API_KEY>`
- **开发网** `wss://laserstream-devnet.helius-rpc.com/?api-key=<API_KEY>`

<Note>Websockets 有一个 10 分钟的不活动计时器；强烈建议实施健康检查并每分钟发送 ping 以保持 websocket 连接活跃。</Note>

## 通知格式

结果是最新的根槽号。

<RequestExample>

```json Request
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "rootSubscribe"
}
```

</RequestExample>

<ResponseExample>

```json Response
{
  "jsonrpc": "2.0",
  "result": 0,
  "id": 1
}
```

```json Notification
{
  "jsonrpc": "2.0",
  "method": "rootNotification",
  "params": {
    "result": 42,
    "subscription": 0
  }
}
```

</ResponseExample>
