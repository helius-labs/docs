**新功能**: `getAccountInfo` 现在支持 `changedSinceSlot` 参数用于增量更新。当指定时，该方法仅返回在给定槽号或之后修改的账户。对于存在但自指定槽号以来未更改的账户，响应将包含 `status: "unchanged"`。
</Info>
