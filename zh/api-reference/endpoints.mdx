## Solana RPC 端点

标准的高性能 RPC 端点，提供完整的 Solana API 兼容性，并增强了可靠性和吞吐量。这些端点支持所有 Solana JSON-RPC 方法，现在对于所有付费计划**默认使用 staked 连接**，提供最佳的交易着陆率：

- **主网**: `https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY`
- **开发网**: `https://devnet.helius-rpc.com/?api-key=YOUR_API_KEY`

<Note>
**Staked Transactions Are the New Default**

通过 Helius 发送的交易将自动使用 staked 连接，适用于所有付费计划，为您提供最高的交易着陆率，并确保您的交易在速度和可靠性方面得到优化。无需更改代码。
</Note>

## 安全 RPC 端点

这些 URL 专门进行了掩码处理，并在 5 TPS 下进行 IP 速率限制，使其可以安全地直接从前端应用程序使用，而不会暴露您的 API 积分：

- **主网**: `https://abc-456-fast-mainnet.helius-rpc.com`
- **开发网**: `https://123-xyz-fast-devnet.helius-rpc.com`

## Staked Connection 端点（已弃用）

<Warning>
**已弃用**: 质押端点 (`staked.helius-rpc.com`) 已被弃用。请改用常规端点 (`mainnet.helius-rpc.com`)，该端点现在默认使用质押连接，适用于所有付费计划。
</Warning>

- **费用**: 每个 `sendTransaction` 请求1个信用点（从10个信用点减少）
- **自动优化**: 所有交易通过优化的网络基础设施发送（亚洲、欧洲和北美）
- **更高的着陆率**: 在市场拥堵期间保证访问质押连接
- **无需代码更改**: 现有应用程序自动受益于质押连接

## Solana Websocket 端点

标准 WebSocket 端点，根据 Solana WebSocket 协议提供实时数据流。这些端点支持所有标准的 Solana WebSocket 订阅方法，包括账户、程序、签名、槽位和日志订阅：

- **主网**: `wss://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY`
- **开发网**: `wss://devnet.helius-rpc.com/?api-key=YOUR_API_KEY`

### 关键特性

- **支持的订阅**: `accountSubscribe`, `programSubscribe`, `signatureSubscribe`, `slotSubscribe`, `logsSubscribe`，等等
- **协议兼容性**: 完全兼容Solana的标准WebSocket API
- **最佳用途**: 需要监控账户变动、跟踪交易确认或接收区块链活动实时更新的应用程序
- **连接管理**: 每30-60秒实施一次定期ping以保持活动连接

## Geyser 增强型 Websocket 端点

增强型 WebSockets 提供比传统 RPC WebSockets 更快的响应时间，并直接向您的应用程序提供实时交易和账户更新：

- **主网**: `wss://atlas-mainnet.helius-rpc.com/?api-key=YOUR_API_KEY`
- **开发网**: `wss://atlas-devnet.helius-rpc.com/?api-key=YOUR_API_KEY`

### 关键特性

- **适用于**: 仅限商业和专业计划（与Yellowstone Geyser不同，后者需要专用节点）
- **订阅方法**: 支持 `transactionSubscribe` 和 `accountSubscribe`
- **连接管理**: Websockets有10分钟不活动计时器；实施健康检查并每分钟发送ping以保持连接
- **当前状态**: 处于测试阶段，偶尔可能会出现数据延迟
