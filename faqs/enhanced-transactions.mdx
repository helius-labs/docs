## Using Enhanced Transactions

<Accordion title="What is the Enhanced Transactions API?">
It returns parsed transaction data in a human-readable format, lets you parse single or multiple transactions, and fetch full history for an address.
</Accordion>

<Accordion title="What types of transactions can be parsed?">
NFT, Jupiter, Pump, and SPL transactions. Don't rely on it for other types.
</Accordion>

## Authentication

<Accordion title="How do I authenticate API requests?">
Add your Helius API key as the `?api-key=` query parameter.
</Accordion>
