The Helius Digital Asset Standard (DAS) API provides powerful tools for reading and querying both NFT and token data on Solana. This comprehensive guide covers how to work with all types of Solana assets.

## Price Data for Jupiter Verified Tokens

For tokens that are verified on Jupiter, you can access real-time price data through the `getAsset` method:

```typescript
const response = await fetch("https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    jsonrpc: "2.0",
    id: "1",
    method: "getAsset",
    params: {
      id: "DezXAZ8z7PnrnRJjz3wXBoRgixCa6xjnB7YaB1pPB263", // Bonk token mint address
      displayOptions: {
        showFungibleTokens: true
      }
    },
  }),
});
const data = await response.json();
console.log(data);

// Calculate market cap
const marketCap = data.result.token_info.price_info.price_per_token * data.result.token_info.supply;
console.log(`Market Cap: $${marketCap.toLocaleString()}`);
```

## Working with NFTs and Digital Collectibles

### Getting a Single Asset

Retrieve comprehensive data for a specific NFT or token using `getAsset`:

```typescript
const response = await fetch('https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY', {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    jsonrpc: "2.0",
    id: "1",
    method: "getAsset",
    params: {
      id: "F9Lw3ki3hJ7PF9HQXsBzoY8GyE6sPoEZZdXJBsTTD2rk",
    },
  }),
});
const data = await response.json();
console.log(data);
```

### Querying Assets by Owner

Find all assets owned by a specific wallet:

```typescript
const response = await fetch("https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    jsonrpc: "2.0",
    id: "1",
    method: "getAssetsByOwner",
    params: {
      ownerAddress: "86xCnPeV69n6t3DnyGvkKobf9FdN2H9oiVDdaMpo2MMY",
      page: 1,
      limit: 10,
    },
  }),
});
const data = await response.json();
console.log(data);
```
