## Standard Archival Methods

Access historical blockchain data with these standard RPC methods. All archival calls cost 10 credits. [View detailed pricing →](/billing/plans-and-rate-limits#credit-system)

<Tabs>
  <Tab title="Block Data">
    Query historical block information and metadata.
    
    <AccordionGroup>
      <Accordion title="getBlock" icon="cube">
        Retrieve complete block data including all transactions and metadata for a specific slot.
        
        **Use cases:** Block explorers, audit tools, historical analysis
        
        [View getBlock guide →](/rpc/guides/getblock)
      </Accordion>
      
      <Accordion title="getBlocks" icon="cubes">
        Get a list of confirmed blocks between two slots.
        
        **Use cases:** Gap detection, block range analysis, indexing
        
        [View getBlocks guide →](/rpc/guides/getblocks)
      </Accordion>
      
      <Accordion title="getBlockTime" icon="clock">
        Get the estimated production time of a block as Unix timestamp.
        
        **Use cases:** Time-based analytics, event correlation
        
        [View getBlockTime guide →](/rpc/guides/getblocktime)
      </Accordion>
    </AccordionGroup>
  </Tab>
  
  <Tab title="Transaction History">
    Access historical transaction data and signatures.
    
    <AccordionGroup>
      <Accordion title="getTransaction" icon="file-lines">
        Retrieve full transaction details for a specific signature.
        
        **Use cases:** Transaction verification, detailed analysis, debugging
        
        [View getTransaction guide →](/rpc/guides/gettransaction)
      </Accordion>
      
      <Accordion title="getSignaturesForAddress" icon="signature">
        Get confirmed signatures for transactions involving an address.
        
        **Use cases:** Basic transaction history, signature collection
        
        <Note>
        For advanced filtering and sorting, use [`getTransactionsForAddress`](/rpc/gettransactionsforaddress) instead.
        </Note>
        
        [View getSignaturesForAddress guide →](/rpc/guides/getsignaturesforaddress)
      </Accordion>
      
      <Accordion title="getInflationReward" icon="coins">
        Get inflation reward for a list of addresses for an epoch.
        
        **Use cases:** Validator rewards, staking analytics
        
        [View getInflationReward guide →](/rpc/guides/getinflationreward)
      </Accordion>
    </AccordionGroup>
  </Tab>
</Tabs>
