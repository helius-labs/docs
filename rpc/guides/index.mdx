# RPC Method Guides

Welcome to the RPC Method Guides section! Here you'll find a growing collection of practical, bite-sized guides focused on specific Solana RPC methods. 

Each guide aims to answer the question: "How do I use `[RPC Method]` to achieve `[a common task]`?"

Whether you're fetching account data, sending transactions, or querying block information, these guides are designed to get you started quickly with clear examples and explanations.

## Available Guides

Browse through the guides to understand the most popular use cases for each RPC call:

*   **Account Information:** Learn how to retrieve details about accounts, their balances, and associated token data.
*   **Block & Slot Information:** Understand how to fetch data about blocks, slots, and epoch schedules.
*   **Transaction Handling:** Discover how to send transactions, simulate them, and retrieve their status.
*   **Network Information:** Get details about the cluster, node health, and more.
*   **WebSocket Subscriptions:** Learn how to subscribe to real-time updates for accounts, blocks, logs, and more.

We are continuously adding new guides. If you have a specific use case in mind or a suggestion for a new guide, please let us know!
