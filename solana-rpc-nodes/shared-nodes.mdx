## How to use Helius Shared RPCs

<Steps>
  <Step title="Generate an API key">
    Create an API key from the [dashboard](https://dashboard.helius.dev/).
  </Step>
  
  <Step title="Use the appropriate endpoint">
    To make [standard RPC calls](/rpc/http) use these endpoints:
    
    * **Mainnet** – `https://mainnet.helius-rpc.com?api-key=YOUR_API_KEY`
    * **Devnet** – `https://devnet.helius-rpc.com?api-key=YOUR_API_KEY`
    
    Add your API key to the endpoint URL. The complete endpoints are also available in the [Endpoints](https://dashboard.helius.dev/endpoints) section of your dashboard.
    
    ![dashboard.helius.dev/endpoints](/images/dashboard-endpoints.png)
  </Step>
  
  <Step title="Configure access controls (optional)">
    Our platform offers controls to customize your RPC access patterns—for instance, restricting requests to specific IPs or domains. These configurations can be managed directly in the [dashboard](https://dashboard.helius.dev/endpoints).
    
    ![Configure RPC Access Control Rules in the Endpoints section of your dashboard.](/images/dashboard-endpoints-access-control.png)
  </Step>
</Steps>
