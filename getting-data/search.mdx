## Overview

This method will return assets based on the custom search criteria passed in. This can define compressed, regular NFTs, and fungible tokens. This method is optimal for most custom use cases, such as token gating.

<Callout type="info">
  The **`page`** parameter in the request starts at **1**.
</Callout>

<Card title="Search Assets Endpoint" icon="magnifying-glass">
  <ParamField path="/" method="POST">
    ```json
    {
      "jsonrpc": "2.0",
      "id": "my-id",
      "method": "searchAssets",
      "params": {
        "ownerAddress": "86xCnPeV69n6t3DnyGvkKobf9FdN2H9oiVDdaMpo2MMY",
        "tokenType": "fungible"
      }
    }
    ```
  </ParamField>
  <ResponseField name="result" type="object">
    Response will include assets that match the search criteria
  </ResponseField>
</Card>

## Fungible Token Extension

Fungible tokens will include their associated information (decimals, supply, and price). Token22 tokens are also supported, and their extensions are parsed. We also display native balances if you enable the `showNativeBalance` flag.

The options for `tokenType` include:

<Properties>
  <Property name="fungible" type="string">
    Returns all fungible tokens
  </Property>
  <Property name="nonFungible" type="string">
    Returns all NFTs (compressed and regular NFTs)
  </Property>
  <Property name="regularNFT" type="string">
    Returns only the regular NFTs
  </Property>
  <Property name="compressedNFT" type="string">
    Returns only the compressed NFTs
  </Property>
  <Property name="all" type="string">
    Returns all the tokens
  </Property>
</Properties>

When using the `tokenType` as a filter, it is mandatory to also include the `ownerAddress` filter.

## Inscriptions & SPL-20

You can optionally display inscription and SPL-20 token data with the `showInscription` flag. You can learn more about inscriptions and SPL-20 [here](https://x.com/nick_pennie/status/1727877242045542858).

<Note>This is an experimental feature.</Note>

<Callout type="warning">
  The Helius API does not verify SPL-20 tokens. Trade at your own risk. For more information, please use the [validator tool](https://www.spl20.io/) or join the [Libreplex Discord channel](https://discord.gg/kTRGwcXx).
</Callout>

## Code Examples

<CodeGroup>
  <CodeBlock title="Searching for all Fungible Tokens in a wallet" language="javascript">
```javascript
const url = `https://mainnet.helius-rpc.com/?api-key=<api_key>`

const searchAssets = async () => {
  const response = await fetch(url, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      jsonrpc: '2.0',
      id: 'my-id',
      method: 'searchAssets',
      params: {
        ownerAddress: '86xCnPeV69n6t3DnyGvkKobf9FdN2H9oiVDdaMpo2MMY',
        tokenType: 'fungible',
      },
    }),
  });
  const { result } = await response.json();
  console.log("Search Assets: ", result);
};
searchAssets();
```
  </CodeBlock>

  <CodeBlock title="Showing native balance and token info" language="javascript">
```javascript
const url = `https://mainnet.helius-rpc.com/?api-key=<api_key>`

const searchAssetsTokenInfo = async () => {
    const response = await fetch(url, {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
        },
        body: JSON.stringify({
            jsonrpc: '2.0',
            id: 'my-id',
            method: 'searchAssets',
            params: {
                ownerAddress: '86xCnPeV69n6t3DnyGvkKobf9FdN2H9oiVDdaMpo2MMY',
                tokenType: 'fungible',
                displayOptions: {
                    showNativeBalance: true,
                },
            },
        }),
    });

    const { result } = await response.json();

    result.items.forEach(item => {  // for each item in the result, log the token_info
      console.log(item.token_info);
    });

    console.log("Native Balance: ", result.nativeBalance); // logging native balance
};

searchAssetsTokenInfo();
```
  </CodeBlock>
