Transactions represent all state changes on Solana. Helius provides multiple ways to read transaction data, from raw RPC methods to enhanced transaction APIs.

## Getting a Single Transaction

Retrieve transaction details using the standard RPC method:

```typescript
const response = await fetch("https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY", {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    jsonrpc: '2.0',
    id: '1',
    method: 'getTransaction',
    params: [
      '5brUqZacYhjphkFufZsdJoWA3ACTDh7wX5o4ewHiUGPK5zrLWExsFJb1o83q4SidvgegANZMFqocC4cPyymSEdyt',
      {
        encoding: 'json',
        maxSupportedTransactionVersion: 0
      }
    ]
  })
});
const data = await response.json();
console.log('Transaction data:', data);
```

## Enhanced Transaction History

For richer transaction history with parsed data:

```typescript
const API_KEY = 'YOUR_API_KEY';
const WALLET_ADDRESS = 'DjPi9wgUDTtYuuL6ZNvhgKJgC8gHaRKtHq6K5ESWbGKt';

const response = await fetch(`https://api.helius.xyz/v0/addresses/${WALLET_ADDRESS}/transactions?api-key=${API_KEY}&limit=10`, {
  method: 'GET',
});
const transactions = await response.json();
console.log('Enhanced transaction history:', transactions);
```

## Getting Multiple Transactions

Batch retrieve multiple transactions:

```typescript
const HELIUS_API_URL = 'https://api.helius.xyz';
const API_KEY = 'YOUR_API_KEY';

const response = await fetch(HELIUS_API_URL + '/v0/transactions', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${API_KEY}`
  },
  body: JSON.stringify({
    transactions: [
      '5rxRt2GVpSUFJTqQ8YtYm3m2T47bzWNKuXS4FmtfKW3o2esgxtdgqUJpgZbc5wfGwmJvLq9jY8Wdbz7q3r8U4p96',
      '4Ej2KZ9mGQQQr5MGNAKsAXxbzALDzEd7nuaWJ3iHxzNr4VNyiLKj8xwCKzS5xLKDqpGbpdeiZu4c4bCSTQRrPGCj',
      '2Rs7PE8rBj9ToBgv2pJbgzbURcJ5JhYcSwVCPJr3cSBjYxU5oFtUE2JkCPfA5hjYQqLk1gCuprMpVQPJpizwTgj1'
    ]
  })
});
const enhancedTxs = await response.json();
console.log('Multiple transactions:', enhancedTxs);
```

## Transaction Status Checking

Check the status of multiple transactions:

```typescript
const response = await fetch("https://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY", {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    jsonrpc: '2.0',
    id: '1',
    method: 'getSignatureStatuses',
    params: [
      [
        '5rxRt2GVpSUFJTqQ8YtYm3m2T47bzWNKuXS4FmtfKW3o2esgxtdgqUJpgZbc5wfGwmJvLq9jY8Wdbz7q3r8U4p96',
        '4Ej2KZ9mGQQQr5MGNAKsAXxbzALDzEd7nuaWJ3iHxzNr4VNyiLKj8xwCKzS5xLKDqpGbpdeiZu4c4bCSTQRrPGCj'
      ],
      {
        searchTransactionHistory: true
      }
    ]
  })
});
const statuses = await response.json();
console.log('Transaction statuses:', statuses.result);
```

## Real-time Transaction Monitoring

For real-time transaction updates, use WebSockets:

```typescript
const HELIUS_WS_URL = 'wss://mainnet.helius-rpc.com/?api-key=YOUR_API_KEY';
const ws = new WebSocket(HELIUS_WS_URL);

ws.onopen = () => {
  console.log('WebSocket connection established');
  ws.send(JSON.stringify({
    jsonrpc: '2.0',
    id: 1,
    method: 'signatureSubscribe',
    params: [
      '5rxRt2GVpSUFJTqQ8YtYm3m2T47bzWNKuXS4FmtfKW3o2esgxtdgqUJpgZbc5wfGwmJvLq9jY8Wdbz7q3r8U4p96',
      {
        commitment: 'finalized'
      }
    ]
  }));
  console.log('Subscription request sent');
};

ws.onmessage = (event) => {
  const response = JSON.parse(event.data);
  console.log('Transaction update received:', response);
};

// Example: closing connection after 60 seconds
setTimeout(() => {
  ws.close();
  console.log('WebSocket connection closed');
}, 60000);
```
